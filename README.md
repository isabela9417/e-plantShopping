## **Overview**

Paradise Nursery is an online plant shop where users can:

- Browse a variety of house plants.
- Add plants to a shopping cart.
- Adjust quantities or remove items from the cart.
- View a dynamic total cost.
- Navigate easily between pages.

This project highlights **React functional components**, **React Hooks**, and **Redux Toolkit** integration.

---

## **Features**

1. **Landing Page**
   - Welcome message and button linking to the Product Listing page.

2. **Navigation Bar**
   - Links to Landing, Products, and Cart pages.
   - Cart icon dynamically displays the number of items in the cart.

3. **Product Listing Page**
   - Plants displayed in cards with image, name, description, price, and Add to Cart button.
   - Sections for plant categories like “Aromatic Plants” and “Medicinal Plants”.

4. **Shopping Cart**
   - Each plant type has a card showing:
     - Thumbnail
     - Unit price
     - Total price
     - Quantity controls (increase/decrease)
     - Delete button
   - Total cart cost dynamically updates.
   - Continue Shopping and Checkout buttons.

5. **Responsive Design**
   - Mobile-friendly layout.
   - Styled with CSS for a clean and modern look.
